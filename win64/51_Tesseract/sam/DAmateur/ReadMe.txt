Transformator4D Version 2

Программ моделирования трехмерных проекций четырехмерных объектов
Отличие от версии в том что аффинные преобразования возможно создавать для каждого полигона отдельно


Оси разных цветов в порядке RGBV

----------
Управление.
Мышь
Левая кнопка - управление камерой
Правая кнока - приближение/отдаление камеры

----------
Клавиатура
Движение точки взгляда
по оси X - Q,A
по оси Y - W,S
по оси Z - E,D

Q/A W/S E/D R/F - управление четырехмерным вектором проекции. СОответсвеноо x,y,z,q увеличить/уменьшить


X - показать/скрыть координатную сетку
C - показать/скрыть оси
V - показать/скрыть объект


O/L - толщина линий фигуры

---------
Для работы необходимы непустые файлы 
acrTrans1FA.txt  -  файл четырехмерного объекта аффинных преобразований его полигонов
