Запуск программы осуществляется из командной строки командой: Cmos.exe.

В результате в верхнем левом углу появятся три строки с общей информацией о программе.
По нажатию любой клавиши на дисплее отображается первая из четырёх страниц с 
информацией по текущим значениям ячеек памяти CMOS. 
Структура страницы приведена ниже:

Адрес – поле, где указаны адреса ячеек памяти CMOS в шестнадцатеричном виде.
Значение – поле, где указаны текущие значения в соответствующих ячейках CMOS.
Бит – высвечивается побитовая структура ячейки. Существует не у каждого адреса,
           а только у тех, где нужна побитовая раскладка ячейки CMOS.
Пояснение – краткий комментарий о назначении данной ячейки CMOS.

Переключение страниц осуществляется клавишами PgUp/PgDn (Вверх / Вниз).
Нижняя строчка на странице отображает  фунциональные клавиши и  соответствующие им 
действия.
F2 – Запись содержимого всей CMOS в файл CMOS.BAK
F3 – Восстановление всей CMOS из файла CMOS.BAK
F4 – Редактирования выбранной ячейки CMOS
F10 – Выход из программы 


По нажатию F2 программа сохранит содержимое CMOS.BAK в файл и выдаст на экран 
сообщение:
«CMOS скопирована в файл. Нажмите любую клавишу».

По нажатию F3 программа восстановит содержимое CMOS.BAK из файла и выдаст на экран 
сообщение:
«CMOS скопирована из файла. Нажмите любую клавишу».

По нажатию F4 программа сохранит содержимое CMOS в файл CMOS.SAV и выдаст на экран 
сообщение:
«CMOS скопирована в файл. Нажмите любую клавишу»
По нажатию любой клавиши – запрос на ввод двузначного шестнадцатеричного числа:
«Введите редактируемый адрес в hex форме - ». Надо ввести адрес ячейки значение которой вы 
собираетесь редактировать. По нажатию ESC запрос исчезает. После ввода числа на экране 
появляется строка: «o.k.  Press any key».
По нажатию любой клавиши - запрос на ввод двузначного шестнадцатеричного числа:
«Введите значение в hex форме - ». Здесь нужно ввести новое значение редактируемой ячейки. 
По нажатию ESC запрос исчезает.
После ввода числа на экране появляется строка: «o.k.  Press any key». Нажмите любую клавишу 
и вы опять перейдете в режим просмотра дампа Cmos.
